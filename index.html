TicTacToe


 


     
Rules
  1. Player can enter symbols which they want.
  2. Valid symbols means both symbols has to be different and should not be in ['',' '].
  3. Player has to click on start button to start the game.
  4. After clicking on start they get a board to play.
  5. They has to click on the cell in which they want to place their symbol.
  6. If the whole board is full and none of the players has won then "No one Won" message is displayed, otherwise winning message is displayed.
  7. Winning condition is all cells in row or column or diagonal should have same symbol.
